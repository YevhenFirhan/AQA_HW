
"GIT Pull" - a command that is used to extract data from a remote repo to your local repo, data is not only extracted but also merged with your local repo data. 
	A command itself is a combo of command "git fetch"(that downloads data from a remote repo to your local repo, but doesn't connect it with local data) and "git merge" command, that joins 2 or more brances data into 1 new branch. 
 	 - git pull "the name of the remote repo"
	 - git merge "the name of the branch we want to merge in with the branch we're currently in".
